What is a newton fractal?

A Newton fractal is created by performing a newton approximation on every point of a complex function and displaying the final result based on the initial and final Z values

Newton's method is defined by the iteration of the following series:

Z -> Z - F(Z)/F'(Z)

Where F(Z) is the function (in our case a complex function), and F'(Z) is the first derivative of the function

In this program the function is defined by:

F(Z) = (Z - C1)(Z - C2)(Z - C3)

where C1, C2, and C3 are defined by the interactive zeros
